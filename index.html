Services

Mobile app development

I can create Android and iOS (iPhone and iPad) apps for smartphones and tablets. This includes native apps as well as PhoneGap/Cordova/HTML5 apps.

Search Engine Optimisation (SEO)

I can optimise your web content to improve your Google search ranking to drive more traffic to your site and increase conversion rates.

Cloud hosting migrations and services

I can create web and mobile solutions that take advantage of scalable cloud technologies such as Amazon Web Services (AWS), Heroku, Google Cloud Platform, Google Apps (Google Mail, Calendar, Drive, Docs) and Digital Ocean.

General software development

I’m a versatile programmer who can work on projects ranging from web services to desktop apps. I primarily use JavaScript, Node.js, Python, Java, OCaml, C and PHP. As I know a diverse range of languages I can quickly adapt to new technologies.

Portfolio

A selection of web and mobile projects I’ve worked on.

Fresco mobile app

I independently designed and created Fresco for Android, an easy to use yet feature packed digital painting app for phones and tablets. The app features layers, customisable brushes, image filters and more. It has sold over 10K copies and has over 500K free downloads. Technologies used: Android, C, Java.

Cloudreach website

I was involved in the corporate website for cloud computing specialists Cloudreach. My role included providing a solution for multilingual support, cloud hosting setup, SEO, analytics for visitor behaviour, optimising performance and WordPress theme customisation. Technologies used: Heroku, AWS, WordPress, PHP, jQuery.

AV Search mobile app

I was lead developer on the team that made AV Search for iOS and Android for the National Apprenticeship Service (a UK government agency). The app integrates several online web services to make it easy to search and apply for apprenticeships in England. Technologies used: PhoneGap/Cordova, JavaScript, Heroku, Python, Flask, CouchDB.

Old Low Light museum game

In collaboration with production company Freak Works, I coded a large screen touch input game for the Old Low Light nautical museum. I worked closely with the museum and artists to create a fun and educational experience that taught visitors how light houses are used by sailors to steer to shore. Technologies used: Heroku, Cocos2d, JavaScript.
