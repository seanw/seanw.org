About me

A short summary of my career so far and my technical skills.

Career background

I grew up in Edinburgh and have been passionate about coding since an early age. I attended The University of Edinburgh where I was top of my year earning a Software Engineering degree and completed a PhD in Software Verification. I then worked as a computer science researcher before joining a cloud computing consultancy as a software engineer. I now work for myself as an app developer and have been successfully doing so for several years. I have a mixture of regular clients with ongoing projects and new clients with shorter term projects while I also work on my own side projects.

Development skills

I’m a full stack developer who has worked in a large range of languages and frameworks including JavaScript (TypeScript, Node, Express, Vue, Angular), Python (Django, Flask), Java (Swing), PHP (WordPress, CodeIgniter, Laravel), C, C++, OCaml, JAMstack (with Hugo, Jekyll) and Coq. I can easily pick up new skills because of my diverse background. I tend to deploy with cloud based services like AWS, Heroku and Netlify with scalable storage solutions like PostgreSQL, S3, DynamoDB, Firebase and Firestore. I love coding and I’m passionate about employing modern development best practices such as Continuous Integration and software testing. I’m practical in using whichever technologies I think works best for a project to stay within budget and reduce complexity.

Past projects

A selection of web and mobile apps that show the diversity of projects I can help with.

Checkbot - Website auditor Chrome extension

I’m the creator of Checkbot, a Chrome extension that checks if your website follows 50+ SEO, speed and security web best practices. I independently developed the extension along with its custom subscription payment solution and I’m responsible for the website, web design and the comprehensive web best practices guide that accompanies the project. Checkbot has a 4.9 out of 5 average user rating and has over 15K active users along with paying customers.

Technologies: Chrome, browser extension, JavaScript, TypeScript, Paddle, Netlify, Node, Express, Firebase, Firestore, Firebase Cloud Functions

Fresco - Android painting app

I independently designed and created Fresco for Android, an easy to use yet feature packed digital painting app for Android phones and tablets. The app features layers, customisable brushes and image filters with a focus on usability. The code was highly optimised for devices with limited memory and processing power. It has sold over 10K copies and has over 500K free downloads.

Technologies: Android, C, Java, JNI

AV Search - iOS & Android apprenticeship search

I was lead developer on the team that made the AV Search mobile app for the National Apprenticeship Service (a UK government agency). The publicly available app lets you search and apply for apprenticeships in the UK and works on iOS and Android. This included writing a web service that lets you search through live apprenticeship data that’s synchronized from another database. A big aspect of the project was managing the complexity of multiplatform development while keeping an eye of development costs.

Technologies: PhoneGap/Cordova, JavaScript, Backbone, Android, iOS, iPhone, Heroku, Python, Flask, CouchDB, Bamboo

Old Low Light - Educational touchscreen game

In collaboration with production company Freak Works, I coded a large screen touch input game for the Old Low Light nautical museum. I worked closely with the museum to create a game design that taught players how light houses are used by sailors to steer to shore. I then worked with graphic designer Andrei Staruiala to create an inviting faux 3D gaming experience that worked within the budget we had.

Technologies: Heroku, Cocos2d, JavaScript, Windows

Cloudreach - Corporate website

I was the core developer responsible for maintaining and enhancing the corporate website for cloud computing specialists Cloudreach who now have revenue of over $1 billion. My role included adding a complete multilingual solution, displaying content that dynamically changed based on the interests of the user, SEO auditing and optimisations, Salesforce integration, cloud hosting setup, analytics for visitor behaviour, optimising performance, theme customisation and creating product landing pages.

Technologies: Heroku, AWS, WordPress, PHP, jQuery, Salesforce, PostgreSQL, S3
