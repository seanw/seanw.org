About me

Background

I grew up in Edinburgh and attended The University of Edinburgh where I earned a PhD in Software Testing and a Software Engineering degree. I then worked as a computer science researcher before joining a cloud computing consultancy as a software development consultant. I went on to work for myself and have been successfully doing so now for around 4 years.

Development skills

I’m a full stack developer who has worked in a large range of programming languages such as JavaScript (TypeScript, Node, Express), Python (Django, Flask), OCaml, Java (Swing), C, C++, PHP (WordPress, CodeIgniter, Laravel). I tend to deploy with cloud based services like AWS, Heroku, Netlify and Firebase. I employ modern developer best practices such as Continuous Integration and software testing. I’m practical and will use whichever technologies I think will work best for a project.

How I can help you

I can design, develop and deliver software solutions for you in the following areas.

Web apps

I can create websites, web apps and web services that support both mobile and desktop. My skills for this include HTML5, CSS3, JavaScript (Node, TypeScript, AngularJS, Vue, jQuery), Python (Flask, Django), Java, PHP (WordPress), Jekyll, Hugo.

Mobile apps

I can create Android and iOS (iPhone and iPad) apps for smartphones and tablets. My skills for this include Java and C for native apps and PhoneGap/Cordova for HTML5 hybrid apps.

Interactive slideshows, games & kiosk experiences

I can create games and interactives like those you’d see at exhibitions and museums including visitor guestbooks, interactive maps and games. This can include working with Android, iOS, Windows and large touchscreen monitors.

Team workflow & software quality improvements

I can oversee projects and provide training for developers to increase productivity and lower defects. This includes helping with staging environment setup, test suites, code reviews, source control, planning boards (Trello and JIRA) and Continuous Integration (Jenkins, Bamboo).

Cloud hosting & migrations

I can help you take advantage of scalable and affordable cloud technologies such as Amazon Web Services (AWS), Heroku, Firebase, Google Cloud Platform, Google Apps (Google Mail, Calendar, Drive, Docs) and Digital Ocean.

Search Engine Optimisation (SEO)

I can perform on-page SEO audits of websites and have the expertise to fix any issues found to help to improve your Google search rank. This includes helping with website security (such as setting up HTTPS) and optimising page speed.

Past projects

A selection of web and mobile apps I've worked on to show the diversity of projects I can help with.

Checkbot - Website auditor Chrome extension

I’m the creator of Checkbot, a Chrome extension that checks if your website follows 50+ SEO, speed and security web best practices. I independently developed the extension and I’m responsible for the website, web design and the comprehensive web best practices that accompanies the project. Checkbot has a 4.9 out of 5 average user rating and has over 15K active users along with paying customers.

Technologies used: Chrome, browser extension, JavaScript, TypeScript, Paddle, Netlify, Node, Express, Firebase

Fresco - Android painting app

I independently designed and created Fresco for Android, an easy to use yet feature packed digital painting app for phones and tablets. The app features layers, customisable brushes, image filters and more. It was highly optimised to run on lowered end Android phones with limited processing power and memory. It has sold over 10K copies and has over 500K free downloads.

Technologies used: Android, C, Java, JNI

AV Search - iOS & Android apprenticeship search

I was lead developer on the team that made the AV Search mobile app for the National Apprenticeship Service (a UK government agency). The app lets you search and apply for apprenticeships in the UK and works on iOS and Android. The project included managing the complexities of multiplatform support and writing a web service that lets you search through apprenticeship data that is periodically pulled in from another database.

Technologies used: PhoneGap/Cordova, JavaScript, Android, iOS, Heroku, Python, Flask, CouchDB, Bamboo

Old Low Light - Educational touchscreen game

In collaboration with production company Freak Works, I coded a large screen touch input game for the Old Low Light nautical museum. I worked closely with the museum and a graphic designer to create a fun and educational experience that taught visitors how light houses are used by sailors to steer to shore.

Technologies used: Heroku, Cocos2d, JavaScript

Cloudreach - Corporate website

I was the core developer responsible for the corporate website for cloud computing specialists Cloudreach (who now have revenue of over $1 billion). My role included providing a solution for multilingual support, cloud hosting setup, SEO optimisations, analytics for visitor behaviour, optimising page speed and theme customisation.

Technologies used: Heroku, AWS, WordPress, PHP, jQuery
