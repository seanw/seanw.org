Triumph Interactive Map

The Triumph Interactive Map is a private web app that will be displayed at a public exhibition and installed on only one computer. It will show a map overlaid with hand selected Instagram posts.

Privacy Policy

No user data is ever entered into the app or shared in any form. The Instagram client created to access the Instragram API will only have one account associated with it for the purpose of downloading public Instagram posts for public display.
